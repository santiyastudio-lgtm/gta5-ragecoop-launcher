# GTA5 RAGECOOP Launcher

![GTA5 RAGECOOP Launcher](assets/github-avatar.png)

Windows GUI launcher для локального кооператива GTA V через RAGECOOP/RageCoop+.

Главная идея: адрес сервера не зашит в программу. У каждого хоста он свой: launcher берёт IPv4 из активного VPN-адаптера и добавляет порт `4499`. Игрок вручную вставляет адрес, который прислал хост.

## Что умеет

- Вкладка `Я Хост`: установка клиента мода в выбранную папку GTA V, запуск сервера, поиск VPN IP, копирование адреса вида `10.x.x.x:4499` или `26.x.x.x:4499`.
- Вкладка `Я Игрок`: выбор папки GTA V, ввод адреса хоста, запись `LastServerAddress` в конфиг RageCoop/RageCoop+, запуск GTA V.
- Поддержка RageCoop+ для GTA V Enhanced.
- Установщик через Inno Setup: меню Пуск, ярлык, uninstall, встроенный выбор VPN/tunnel.

## Важно про сеть

RAGECOOP использует UDP-порт `4499`. Обычный `ngrok tcp` для этого не подходит. Нужен один из вариантов:

- оба игрока в одной VPN-сети: WireGuard, AmneziaVPN, ZeroTier, Tailscale, OpenVPN и похожие;
- или проброшенный UDP-порт `4499` на роутере хоста.

`127.0.0.1:4499` подходит только самому хосту на его компьютере. Другу такой адрес давать нельзя.

## VPN в установщике

В `GTA5CoopSetup.exe` встроены варианты подключения:

- `WireGuard VPN`: setup запускает встроенный MSI-установщик WireGuard.
- `playit.gg tunnel`: setup копирует и запускает встроенный `playit.exe`.
- `Ничего не устанавливать`: если VPN уже настроен вручную.

Для игры через VPN оба игрока должны выбрать один и тот же сетевой способ и быть в одной сети.

## Как пользоваться хосту

1. Установить `GTA5CoopSetup.exe`.
2. Открыть `GTA5 RAGECOOP Launcher`.
3. Включить VPN, где хост и друг находятся в одной сети.
4. Вкладка `Я Хост`.
5. Выбрать папку GTA V.
6. Нажать `Запустить сервер` или `RageCoop+ server`.
7. Скопировать адрес, который показал launcher, и отправить другу.

## Как пользоваться игроку

1. Установить `GTA5CoopSetup.exe`.
2. Открыть `GTA5 RAGECOOP Launcher`.
3. Вкладка `Я Игрок`.
4. Выбрать папку GTA V.
5. Вставить адрес, который дал хост, например `10.x.x.x:4499`.
6. Нажать `Добавить кооператив и запустить GTA 5`.
7. В игре дождаться загрузки в одиночную игру.
8. Открыть меню RageCoop+. Обычно это `F12`; у обычного RAGECOOP может быть `F9`.

## Сборка из исходников

```powershell
python -m pip install -r requirements.txt
python -m PyInstaller GTA5CoopLauncher.spec --noconfirm
python -m PyInstaller GTA5CoopInstaller.spec --noconfirm --clean
.\build_windows_setup.ps1
```

Готовый обычный Windows-установщик появится здесь:

```text
dist\GTA5CoopSetup.exe
```

## Что не входит

Проект не содержит и не должен распространять файлы GTA V. У каждого игрока должна быть своя легальная установленная копия игры.
